All the gadgets I have, and the software I use to make them sing.




Hi-Fi cast for Android. It's not great, but it does work, and this version is free. Finding a media player for Android that plays files over a local network is a bit of a hassle.


Foobar 2000 for Windows 10. This is the media player that all the music nerds recommend apparently. It's a bit tricky to lear, but very powerful.



Winamp for Windows XP, because it kicks the llamas ass.


Network Media Player for the Playstation Vita. It can't play FLAC files natively, but if you set up your media server properly, the server can convert the music into a format it can play. Would I use the Vita to play music? Not often. Do I want to have the ability to play music from there? Always.


Windows Media Player with added media extensions for Windows 7. This is where my music is all ripped. I don't really use this PC for work or playing music, so WMP is all I need. There is a program called Universal Media Server (UMS) that handles all the serving of music to other devices.
